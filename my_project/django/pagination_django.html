Для реализации пагинации в Django можно использовать классы Paginator и Page в модуле django.core.paginator. Ниже приведен пример реализации пагинации на Django: views.py: ```python from django.shortcuts import render from django.core.paginator import Paginator from .models import Post def post_list(request): post_list = Post.objects.all() paginator = Paginator(post_list, 10) # Показывать 10 сообщений на странице page = request.GET.get('page') posts = paginator.get_page(page) return render(request, 'post_list.html', {'posts': posts}) ``` post_list.html: ```html {% extends 'base.html' %} {% block content %}

Список постов

{% if posts %}
    {% for post in posts %}
  • {{ post.title }}
    • {% endfor %}
{% if posts.has_previous %} « первая предыдущая {% endif %} Страница {{ posts.number }} из {{ posts.paginator.num_pages }}. {% if posts.has_next %} следующая последняя » {% endif %}
{% else %}

Нет постов.

{% endif %} {% endblock %} ``` В этом примере мы используем Paginator для разбиения списка постов на страницы по 10 элементов на страницу. Мы также указываем текущую страницу, используя GET-параметр "page". В шаблоне мы выводим список постов и добавляем элементы пагинации для переключения между страницами. Остальное оформление страницы осуществляется стилями CSS.
