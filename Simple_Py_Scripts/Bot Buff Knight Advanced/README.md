Bot Buff Knight Advanced
===========

Бот для игры "Buff Knight Advanced".

Умеет:

  * Кликать на пролетающую фею для получения ништяков
  * Автоматически атаковать

##

Проверялось на разрешениях:
  
  * 2560 x 1440 (Fullscreen)
  * 1980 x 1080 (Windowed)
